## Project Overview
The Erie Street Kitchen website is designed to provide customers with an intuitive browsing experience. It features a hero section, menu display, reservation information, and contact details. The design emphasizes clean visuals, responsive layouts, and easy navigation.

---

## Features
- Fully responsive design for mobile, tablet, and desktop
- Hero section with engaging visuals
- Menu and specials showcase
- Contact and reservation section
- Smooth animations and interactive buttons
